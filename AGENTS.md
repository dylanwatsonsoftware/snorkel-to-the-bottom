- Always run vitest before making changes to the code
- Then add any relevant tests
- Then run vitest again to make sure the tests pass
- If the tests fail, fix the tests and then run vitest again
- If the tests pass, run npm run build to make sure the build works
- If the build fails, fix the build and then run npm run build again
- If the build works, run npm run dev to make sure the game works
- If the game doesn't work, fix the game adding more tests as needed
